# AI for Lawyers — הכשרות AI לעורכי דין

Workshop program, syllabuses, and landing page for **AI for Lawyers** — practical AI training for Israeli law firms, by Nerya Rettig (נריה רטיג).

## What's in this repo

| Path | What it is |
|------|------------|
| `index.html` | The public landing page (Hebrew, RTL) — served via GitHub Pages |
| `syllabus/` | Syllabus documents for every workshop format and audience segment |
| `sources/` | Source materials: the master curriculum and the original landing-page design export |
| `docs/sprint/` | Internal sprint/state tracking for the development process |
| `tests/` | Automated checks for site and syllabus integrity |

## The workshop program

Ten tracks (A–J) covering foundations, everyday legal work, document automation, Microsoft 365 integrations, building tools, personalization (skills & playbooks), agentic workflows, cost management, safety & ethics, and prompting. See [sources/AI-for-Lawyers-Workshop-Curriculum.md](sources/AI-for-Lawyers-Workshop-Curriculum.md) for the full curriculum.

## Website

The landing page is published with GitHub Pages. Custom domain: `lawyers.nerya.io` (see `CNAME`).

### DNS setup (one-time, in Squarespace DNS settings for nerya.io)

Add a CNAME record:

```
Host:  lawyers
Type:  CNAME
Value: vicciz-ceo.github.io
```

Until DNS is added, the site is available at the default GitHub Pages URL.
